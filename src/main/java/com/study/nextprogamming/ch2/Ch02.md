# 문자열 계산기 요구사항 및 실습

- 전달하는 문자를 구분자로 분리한 후, 각 숫자의 합을 구해 반환해야함.
- 쉼표(,) or 콜론(:)을 구분자로 가지는 문자열을 전달하는 경우, 구분자를 기준으로 분리한 각 숫자의 합을 반환함.
- 음수받으면 RuntimeException 발생시키기.
- Custom 구분자 지정 가능, 문자열 // 와 \n 사이에 위치하는 문자를 Custom 구분자로 사용.
